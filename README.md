# Node Js - Streaming

Esse projeto foi um curso da Alura que é direcionado para quem está aprendendo Node e quer aplicar alguns padrões diferentes no projeto, e para isso precisa aprender a manipular arquivos, sobre validação e mbc Repositories.
Veremos como trabalhar com imagens e arquivos de modo geral, utilizando a biblioteca do File System do Node. Além disso, entenderemos como consumir serviços, e que EPIs não serão exclusivas apra browsers.
Vamos construir uma EPI, ou seja, um serviço, dentro da nossa EPI (uma EPI dentro de uma EPI).
Por fim, vamos refatorar um pouco o código que fizemos no último curso e aprender um novo padrão, o de Repositories, que funcionará como um intermediário para conseguirmos separar melhor nosso código.
Bateremos muito na tecla de que cada código deverá ter sua própria responsabilidade, e assim vamos desacoplar nosso sistema, que enfrenta esse problema.
Os pré-requisitos para esse curso são ter feito o último curso de Node, referente à parte 1 do atual, e além disso os pré-requisitos que já existiam para o curso anterior.
Então, fazendo esse curso aprenderemos a organizar melhor o código e pensar melhor na hora da refatoração, pois vamos juntos entrar no código e pensar em como desacoplar alguns itens.
Além disso, saberemos como manipular arquivos e trabalhar melhor com EPIs.
